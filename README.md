# TinyMath
A fixed-point math library, especially for AVR microcontrollers.

In Progress:
- Fixed point to Decimal conversion
- More string conversions (Octal, Hex etc)

Pipeline:
- nPr, nCr
- sin, cos, tan
- log, exp
- pow
- Complex nums
- Matrices & Determinants

Done:
- Basic fixed point impl
- Basic math constructs (+,-,\*,/)
